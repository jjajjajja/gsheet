# gsheet

Easy to use [Google SpreadSheet](https://sheets.google.com) helper for rust

## Features

- Google Sheets API v4 support
- serialize/deserialize from sheet
- CCCC
- DDDD

## Example

Dependencies:

```toml
[dependencies]
gsheets = "0.1.0"
```

Code:

```rust
fn main(){
    println!("Hello Sheets");
}

```
